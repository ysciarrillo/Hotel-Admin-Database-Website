About this project

Black White Hotel is a project for the INF 335 course in Web Server Technologies. The project was completed successfully using the third approach. This is the index page. Each link from the navigation bar leads to any of the required components to view, update, add and delete a record in a hotel database. So this is basically like an administration panel. Every single web form or .php script is written by me. Only the template .css is from a website with free .css designs called www.templatemo.com. :) There are also buttons here that will also navigate to the forms.
